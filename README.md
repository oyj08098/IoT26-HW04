# IoT26-HW04

## Raspberry Pi Web Server using Flask to Control GPIOs

## Objective
Raspberry Pi에서 Flask 웹서버를 실행하고, 웹페이지 버튼으로 GPIO LED 2개를 제어한다.

## Hardware
- Raspberry Pi
- LED 2개
- 저항 220Ω 2개
- Breadboard
- Jumper wires

## GPIO Pins
- LED 1: GPIO 17
- LED 2: GPIO 27

## How to Run
```bash
python3 app.py
